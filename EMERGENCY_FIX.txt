SOLUTION 3: MISSING .htaccess
-----------------------------------------------------------------

1. File Manager → Settings → ☑ Show Hidden Files
2. Look for .htaccess in newwebsitetest folder
3. If missing, create new file named: .htaccess
4. Paste this content:

DirectoryIndex index.html
ErrorDocument 404 /newwebsitetest/index.html

<IfModule mod_rewrite.c>
    RewriteEngine On
    RewriteBase /newwebsitetest/
    RewriteCond %{REQUEST_FILENAME} !-f
    RewriteCond %{REQUEST_FILENAME} !-d
    RewriteRule . /newwebsitetest/index.html [L]
</IfModule>

5. Save with permissions: 644
